# godot-3d-player-controller-v2
A 3D Player Controller for the Godot game engine.
Click [<here>](https://timothycope.com/godot-3d-player-controller-v2/) to try it out using a few example levels.

## Getting Started
Copy the contents of this repo's `/addons/3d_player_controller_v2` folder to the `/addons/3d_player_controller_v2` folder of your project. No need to "activate" it in Godot > Project Settings... > Plugins.
